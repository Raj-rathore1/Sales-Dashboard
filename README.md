# Sales Dashboard (Power BI)

## 🔍 Overview

This project is an interactive **Sales Dashboard** built using Power BI to analyze business performance across regions, sales channels, and products.

The dashboard helps in tracking key metrics, identifying trends, and supporting data-driven decision-making.

---

## Dataset

The dataset includes:

* Sales Amount
* Quantity Sold
* Unit Price & Cost
* Discount
* Region
* Sales Channel (Online / Offline)
* Customer Type
* Sales Representative
* Date (Monthly analysis)

---

## Key KPIs

* 💰 Total Sales
* 📦 Total Quantity Sold
* 📈 Month-over-Month Growth
* 💵 Profit & Profit Margin
* 🏷️ Discount %
* 🛒 Channel Contribution (Online vs Offline)

---

## Dashboard Features

* Sales by Region (Bar Chart)
* Monthly Revenue Trend (Time Series)
* Channel-wise Sales Distribution (Donut Chart)
* KPI Cards (Top Metrics)
* Interactive Filters (Month, Region, Channel)

---

## Tools & Technologies

* Power BI
* DAX (Data Analysis Expressions)
* Excel / CSV

---

## How to Use

1. Download the `.pbix` file
2. Open in **Power BI Desktop**
3. Interact with filters and visuals

---

## Key Insights

* North region generates the highest sales
* Online and Offline channels contribute almost equally
* Sales show fluctuations across months indicating seasonal trends
* Discounts impact overall profitability
